# Assignment 2 — ArrayList and LinkedList

## What is implemented
- `MyList<T>` interface
- `MyArrayList<T>`
- `MyLinkedList<T>` (doubly linked)
- `MyStack<T>`
- `MyQueue<T>`
- `MyMinHeap<T extends Comparable<T>>`

## Restrictions
- No `java.util.*` usage except `Iterator`

## Logical structure choices
- Stack → `MyLinkedList`
- Queue → `MyLinkedList`
- MinHeap → `MyArrayList`

## Run
Compile all `.java` files and run `Main`.
